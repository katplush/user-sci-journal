Setting Goals with Google Analytics

Excited to share my thoughts and findings

Posted by Katerina Plushko on May 27, 2019
“A goal properly set is halfway reached.” – Zig Ziglar

Once I had set up my Google Analytics account and got my self comfortable with it, it was time to add some goal tracking. I decided I would add a Mailing List sign up button and form as well as create PDFs of my posts and incorporate a link to allow visitors to easily download my sage observations.

To do both of these required two different process. I found that creating the destination goal for my Mailing List sign up was a bit of an easier task so lets start there.

Goals

To begin I inserted a modal sign up button onto my index.html page, which pops up a for requesting the visitors name and email and a sign-up or cancel button below that. If the visitor clicks sign-up they are taken to my thanks.html page.

The goal for this type of tracking is set up in the "Goals" section of my Google Analytics Admin page. I selected "Destination" as my goal type since on of the easiest ways to track this goal to completion would be to track if the visitor makes it to the thanks.html page. This way we know not only did they click on the Mailing List button but they actually went through with the sign-up. I set my Goal destination to "Begins with" and URL for the thanks.html page minus the domain portion of the address "/user-sci-journal/thanks".

It took about 24 hrs before I could see the data under the "Conversions" reports tab, once it did I could see exactly how many visitors made it to that thanks.html page.

Events

I also wanted to track track how many visitors would download the PDF files of my posts so I looked into how to track a specific link click. I uploaded PDF versions of my posts and then linked to them with a "Download Post" link placed right at the top under the heading of posts 3-7.

To track clicks on these "Download Post" links I had to add some additional JavaScript and HTML code to all the pages I wanted to track. I found the code needed Track Outbound Links article from from Google Analytics Support. I added the code found in this document, also seen in the image below with initiated the click tracking on the Download links.

Once this code was added it wasn't long before I could see the tracked outbound events showing up under the Behavior reports tab. The Events report provides an overview which provides the number of total events and the events category, which in my case is outbound. The Top Events report can be or the Pages report both show which posts were most popular based on downloads. It was clear to see the my Post 7 was the winner on number of times that download link was clicked. The Event Label was set to the URL of the PDF being downloaded so the reports show the exact URL of each PDF that was selected. An Events Flow report is also available and shows each visitor's session and which PDF were clicked and if they continued on to another page/event after the first.

Summary

Setting up these trackable goals and events doesn't take much time but gives the developer and business owner a unique insight into what visitors are interested in, what they gravitate to, and what grabs their attention. Once a site starts to build up a user base these goals and events allow the developer to better assess where the traffic is coming from, differentiate between which campaigns are attracting more clicks over others and use that data to continue to grow and provide users with content that attracts them.
